# RCRIC — Official Website

Official website for **Rajuk College Robotics & ICT Club (RCRIC)**, founded in 2016 and reformed in 2022 under Rajuk Uttara Model College.

## About

RCRIC is a student-run club focused on robotics, artificial intelligence, and ICT education. Our goal is to give students hands-on experience with real hardware, programming, and engineering challenges.

## Pages

- **Home** — Hero section, activities overview, active project, achievements, about, and gallery
- **Executives** — Current club leadership for the 2026 session

## Tech Stack

- HTML5
- CSS3 
- JavaScript 
- Tailwind CSS via CDN
- Google Fonts 

## File Structure
```

rcric/
├── index.html           # Main page
├── assets/              # Images and media
├── css/
│   ├── variables.css    # Design tokens and CSS variables
│   ├── base.css         # Global resets and typography
│   ├── components.css   # Reusable UI components
│   ├── sections.css     # Page-specific section styles
│   ├── navigation.css   # Header and mobile menu
│   ├── animations.css   # Scroll and transition animations
│   └── preloader.css    # Loading screen
├── js/
│   ├── script.js        # Main page JavaScript
│   └── executives.js    # Executives page JavaScript
├── pages/
│   └── executives.html  # Executives page
└── favicon_io/          # Favicon assets
```

## Running Locally

Open `index.html` in any modern browser. If assets don't load correctly, use the **Live Server** extension in VS Code.
